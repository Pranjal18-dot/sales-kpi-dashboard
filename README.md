# Sales KPI Dashboard – Excel Project

This project contains a dynamic Excel-based KPI Dashboard that tracks quarterly sales performance, revenue, targets, customer growth, and CAC (Customer Acquisition Cost). 
The dashboard updates automatically using a quarter selection dropdown and SWITCH formulas.

---

## Features
- Dynamic Q1–Q4 quarter switching
- Revenue Actual vs Target
- Customer & New Customer analysis
- Average Revenue per Customer
- Customer Acquisition Cost (CAC)
- Previous year vs current year comparison
- Clean and professional dashboard layout
- Automated KPI calculations

---

## Tools & Skills Used
- Microsoft Excel
- SWITCH(), SUM(), AVERAGE(), Data Validation
- Table formatting & KPI design
- Data visualization (Charts)
- PowerPoint for presentation
